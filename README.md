### Installation and Setup Instructions

Clone down this repository. You will need node and npm installed globally on your machine.

### Installation:

npm install

### To Run Test Suite:

npm test

### To Start Server:

npm start

### To View it in the browser:

[http://localhost:3000](http://localhost:3000)
